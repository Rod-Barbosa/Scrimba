- reusing class="columns" for efficiency

```html
        <main>
            <div class="container">
                <div class="columns">
                    <section class="main-content">
                        
                        <h2 class="subhead">Lizards are <span class="white-text">awesome</span></h2>
                        
                        <div class="columns">
                            <div class="col-left">
```
- for extra space between columns just use margin-left, nothing fancy
```css
.col-right {
    margin-left: 25px;
}
```
